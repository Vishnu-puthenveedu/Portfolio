Disha

This project is a Digital Panchayat website that helps villagers access panchayat services online. It allows people to get certificates, file complaints, check welfare schemes, and stay updated with local news in easily and faster way. It developed with PHP, MySQL.

E-Voting

This project is about making a safer and more transparent online voting system using blockchain technology. Traditional voting can have problems like fraud and lack of clarity. Our system uses blockchain to make voting secure, clear, and easy to use. It developed with React Native .

This project is an online auction website where users can buy and sell items by bidding. Sellers list their products, and buyers place bids in real time. The highest bidder wins the item when the auction ends. It developed with React.js, JavaScript, Node.js.
